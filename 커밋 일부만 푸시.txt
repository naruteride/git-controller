# 1. 커밋 내역을 확인하고 원격에 올릴 "마지막 커밋"의 해시(ID) 복사
```bash
git log --oneline
```

## [로그 확인 예시]
```bash
# d4e5f6g 네 번째 커밋 (아직 안 올림)
# a1b2c3d 세 번째 커밋 (아직 안 올림)
# 9f8e7d6 두 번째 커밋 (★ 여기까지 올리고 싶음) <-- 이 해시 복사
# 1a2b3c4 첫 번째 커밋 (당연히 같이 올라감)
```

# 2. 복사한 커밋 해시를 사용하여 특정 시점까지만 푸시
```bash
# 문법: git push origin <커밋해시>:<브랜치명>
git push origin 9f8e7d6:feature/OCBSVR-16014/main
```

# 3. 결과
```bash
# 첫 번째, 두 번째 커밋은 원격에 반영되고, 
# 세 번째, 네 번째 커밋은 로컬에만 안전하게 남음
```
